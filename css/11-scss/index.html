Les bases du sass et scss

  • Installer SASS via npm : "npm install -g sass"
  • pour verifier qu'il est bien installé, lancer "sass" ou "npx sass"
  • Si l'execution des scripts est bloqueé, lancer dans powershell en administrateur "set-executionpolicy unrestricted"
  • Les differenes commandes :
    • sass pathToFile.scss pathToFile.css
    • sass --watch pathToFile.scss pathToFile.css
    • sass --watch sassFolder:cssFolder --stylecompressed
